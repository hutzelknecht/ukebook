# ukebook
A webapp that provides ukulele chords for ukulele lovers.  

The goal is to provide chords and tabs, that can be printed 
out individually or as a combined pdf.

## installation
from any folder run:

    npm install -g sails
    npm install -g gulp
    
and in backend folder run    
    
    npm install
    
in frontend folder run:

    npm install
    gulp
